Welcome to the Ersilia API Documentation
========================================

The **Ersilia API** empowers developers and researchers to integrate state-of-the-art computational biology models into their applications and workflows. By leveraging the Ersilia platform, you can access a wide array of pre-trained models for tasks such as drug discovery, disease modeling, and molecular property prediction.

This documentation provides comprehensive information on the API's functionalities, including detailed descriptions of classes, methods, and modules backed by examples.
